# OpenBuilder
OpenBuilder is an open-source 2D game written in C++, that's currently under development

## Download
Simply clone the repository like so
```
git clone https://github.com/Matt146/OpenBuilder
```

## Compile/Install (Linux)
 - ```cmake .```
 - ```make```
 - ```./a.out```
